## 프로젝트 소개
* 간단한 마우스 피하기 게임이다.
* JavaScript getElement, querySelector를 복습하기 위해 만든 프로젝트이다.
* 스테이지는 3단계까지 있고, 장애물이나 벽에 닿지 않고 도착해야 한다.
